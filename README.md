# Color & Cola - Pedidos
Single Page de lista de pedidos com as opções de Enviar Whatsapp, Copiar Pedido e Gerar PDF.


## Configuração
1. Clonar o projeto.

2. No terminal dentro da pasta raiz do projeto executar o comando 
`npm install` para baixar as dependências.

## Execução
Executar o comando `npm start` e aguardar a pagina http://localhost:5173/ ser aberta automaticamente.

## Alterar numero para envio
Dentro `copy-paste-exodo-app\src\App.tsx` alterar o valor no `phone`.
